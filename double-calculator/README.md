# 🧮 Calculatrice Pro ESIEA

## 📋 Présentation

**Calculatrice Professionnelle** développée par **Euloge Mabiala** - Étudiant à l'ESIEA

Une calculatrice web moderne et polyvalente conçue pour impressionner les recruteurs avec des technologies de pointe et une interface utilisateur exceptionnelle.

## ✨ Fonctionnalités Avancées

### 🎯 Mode Standard
- **Calculatrice double** avec deux opérandes indépendants
- **Opérateurs avancés** : +, -, ×, ÷, ^, √
- **Interface intuitive** avec animations fluides
- **Gestion d'erreurs** robuste

### 🔬 Mode Scientifique
- **Fonctions trigonométriques** : sin, cos, tan
- **Fonctions logarithmiques** : log, ln
- **Fonctions exponentielles** : exp, x^y
- **Constantes mathématiques** : π, e
- **Fonctions spéciales** : factorielle, valeur absolue

### 💰 Mode Financier
- **Calculs de valeur présente** (PV)
- **Calculs de valeur future** (FV)
- **Calculs de paiements** (PMT)
- **Calculs de taux d'intérêt**
- **Calculs de périodes**

### 🔄 Convertisseur d'Unités
- **Longueur** : m, km, cm, mm, mi, yd, ft, in
- **Poids** : kg, g, mg, lb, oz
- **Température** : °C, °F, K
- **Surface** : m², km², cm², ha, ac
- **Volume** : m³, l, ml, gal

## 🚀 Installation et Utilisation

### Prérequis
- Navigateur web moderne (Chrome, Firefox, Safari, Edge)
- Connexion internet pour les CDN

### Installation
1. Clonez le repository :
```bash
git clone https://github.com/votre-username/calculatrice-pro-esiea.git
cd calculatrice-pro-esiea
```

2. Ouvrez `index.html` dans votre navigateur

### Utilisation
1. **Mode Standard** : Calculs basiques avec deux opérandes
2. **Mode Scientifique** : Fonctions mathématiques avancées
3. **Mode Financier** : Calculs financiers professionnels
4. **Convertisseur** : Conversion entre différentes unités
